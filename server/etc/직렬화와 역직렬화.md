## 직렬화란?

직렬화는 데이터 구조나 객체 상태를 저장하거나 전송할 수 있는 포맷(JSON)으로 변환하는 과정

메모리에 잇는 객체를 파일 시스템, 데이터베이스, 또는 네트워크를 통해 전송 후에도 재구성할 수 있음

## 역직렬화란?

직렬화의 반대 과정으로, 데이터를 원래의 객체나 데이터 구조로 복구하는 과정

이걸 통해 저장소나 네트워크를 통해 받은 데이터를 응용 프로그램에서 다시 사용할 수 있는 실제 객체로 변환할 수 있음.

## 직렬화 역직렬화 예시

### 1. 네트워크 통신

객체는 메모리 안에서만 의미가 있어서 그대로 서버에 보낼 수 없음

→ 그래서 객체를 JSON 같은 문자열 형태로 직렬화해서 전송하고 서버에서 받은 응답을 다시 역직렬화해서 객체로 사용함.

### 2. Activity 간 데이터 전달

안드로이드에서 Activity는 서로 다른 컴포넌트라 객체를 직접 공유할 수 없음

→ 이때 객체를 직렬화해서 Intent에 담아서 전달하고, 다음 Activity에서 역직렬화해서 다시 객체로 만듦.

### 3. 데이터 저장

로그인 정보나 사용자 설정처럼 앱을 껐다 켜도 유지해야 하는 데이터는 객체를 그대로 저장할 수 없음

→ 직렬화해서 파일이나 SharedPreferences에 저장하고, 필요할 때 역직렬화해서 사용함.
